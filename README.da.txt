Home Assistant-appen til Homey gør det nemt for dig at tilføje Home Assistant-integrationer til Homey ved at forbinde deres enheder.

Understøttelse af Home Assistant-enheder med Light, Sensor, Binary sensor, Cover og Media player enheder. Home Assistant-appen til Homey lader dig kontrollere disse enheder via Homey Mobile App, Homey Web App samt stemmeassistenter (Google Assistant og Alexa).

Bemærk venligst, at det kun er muligt at tilføje enheder, som er en del af en Home Assistant-enhed, det er ikke muligt at tilføje separate enheder.

Til denne integration skal Homey forbinde til din Home Assistant-instans, enten gennem en statisk lokal IP og port (Homey Pro), port forwarding eller fjernadgang gennem en proxy-service. For mere information, se også dokumentationen om fjernadgang af Home Assistant: https://www.home-assistant.io/docs/configuration/remote/.

L'app Home Assistant per Homey ti permette di aggiungere facilmente le integrazioni di Home Assistant a Homey collegando i loro dispositivi.

Supporto per i dispositivi di Home Assistant con entità Luce, Sensore, Sensore binario, Copertura e Lettore multimediale. L'app Home Assistant per Homey ti consente di controllare queste entità tramite l'app mobile Homey, l'app web Homey e anche assistenti vocali (Google Assistant e Alexa).

Si prega di notare che è possibile aggiungere solo entità che fanno parte di un dispositivo Home Assistant, non è possibile aggiungere entità separate.

Per questa integrazione Homey deve connettersi alla tua istanza di Home Assistant, tramite un IP locale statico e una porta (Homey Pro), inoltro di porta o accesso remoto tramite un servizio proxy. Per ulteriori informazioni, consulta anche la documentazione di Home Assistant riguardo l'accesso remoto: https://www.home-assistant.io/docs/configuration/remote/.

L'application Home Assistant pour Homey facilite l'ajout des intégrations de Home Assistant à Homey en connectant leurs appareils.

Prise en charge des appareils Home Assistant avec des entités de type Lumière, Capteur, Capteur binaire, Volet et Lecteur multimédia. L'application Home Assistant pour Homey vous permet de contrôler ces entités via l'application mobile Homey, l'application web Homey ainsi que les assistants vocaux (Google Assistant et Alexa).

Veuillez noter qu'il est uniquement possible d'ajouter des entités qui font partie d'un appareil Home Assistant, il n'est pas possible d'ajouter des entités séparées.

Pour cette intégration, Homey doit se connecter à votre instance Home Assistant, soit par une IP locale statique et un port (Homey Pro), le transfert de port ou un accès à distance via un service proxy. Pour plus d'informations, consultez également la documentation sur l'accès à distance par Home Assistant : https://www.home-assistant.io/docs/configuration/remote/.

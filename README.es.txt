La app de Home Assistant para Homey facilita agregar integraciones de Home Assistant a Homey conectando sus dispositivos.

Soporte para dispositivos de Home Assistant con entidades de Luz, Sensor, Sensor binario, Cubierta y Reproductor multimedia. La app de Home Assistant para Homey te permite controlar estas entidades a través de la App Móvil de Homey, la App Web de Homey, así como asistentes de voz (Google Assistant y Alexa).

Por favor, ten en cuenta que solo es posible agregar entidades que formen parte de un dispositivo de Home Assistant; no es posible agregar entidades separadas.

Para esta integración, Homey necesita conectarse a tu instancia de Home Assistant, ya sea a través de una IP local estática y puerto (Homey Pro), reenvío de puertos o acceso remoto a través de un servicio proxy. Para más información, consulta también la documentación sobre acceso remoto de Home Assistant: https://www.home-assistant.io/docs/configuration/remote/.

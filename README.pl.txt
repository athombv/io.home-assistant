Aplikacja Home Assistant dla Homey ułatwia dodawanie integracji Home Assistant do Homey, łącząc ich urządzenia.

Wsparcie dla urządzeń Home Assistant z kategoriami Light, Sensor, Binary sensor, Cover i Media player. Aplikacja Home Assistant dla Homey pozwala kontrolować te kategorie za pomocą aplikacji mobilnej Homey, Homey Web App oraz asystentów głosowych (Google Assistant i Alexa).

Należy pamiętać, że można dodawać tylko kategorie będące częścią urządzenia Home Assistant, nie można dodawać oddzielnych kategorii.

Dla tej integracji Homey musi połączyć się z instancją Home Assistant, albo poprzez statyczne lokalne IP i port (Homey Pro), przekierowanie portów lub zdalny dostęp przez usługę proxy. Więcej informacji znajdziesz w dokumentacji na temat zdalnego dostępu przez Home Assistant: https://www.home-assistant.io/docs/configuration/remote/.

Приложение Home Assistant для Homey позволяет легко добавлять интеграции Home Assistant в Homey, подключая их устройства.

Поддерживаются устройства Home Assistant с сущностями Свет, Датчик, Бинарный датчик, Жалюзи и Медиа-плеер. Приложение Home Assistant для Homey позволяет управлять этими сущностями через мобильное приложение Homey, веб-приложение Homey, а также голосовых помощников (Google Assistant и Alexa).

Обратите внимание, что можно добавлять только сущности, которые являются частью устройства Home Assistant, отдельные сущности добавить нельзя.

Для этой интеграции Homey должен подключиться к вашей инстанции Home Assistant через статический локальный IP и порт (Homey Pro), проброс портов или удаленный доступ через прокси-сервис. Для получения дополнительной информации см. документацию о удаленном доступе Home Assistant: https://www.home-assistant.io/docs/configuration/remote/.
